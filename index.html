System Architecture
🏗 Frontend (Mobile App)
1
React Native + Expo
Cross-platform iOS/Android. Use Reanimated for smooth chart animations. Fast refresh for live data.
2
Lightweight Charts (TradingView)
WebView-embedded lightweight-charts for 1m/5m/15m/1H price charts with indicator overlays.
3
WebSocket Client
Persistent WS connection to backend. Auto-reconnect with exponential backoff. Handles live tick data and signal pushes.
⚙️ Backend (Signal Engine)
1
FastAPI (Python) — Core
REST + WebSocket server. Async architecture. Serves signals, levels, market data. Deploy on AWS EC2 t3.medium.
2
Celery + Redis — Task Queue
Runs signal computation every 30s on background workers. Decoupled from API layer. Prevents latency spikes.
3
InfluxDB — Time Series
Stores tick data, OHLCV bars, OI data. Lightning-fast range queries for backtesting engine. Grafana for monitoring.
4
PostgreSQL — State DB
Stores signals, history, user settings, performance metrics. RDS or Supabase.
🔌 Data Sources
★
Zerodha Kite WebSocket (FREE)
Real-time tick data, OHLCV, OI for NIFTY/SENSEX options. Best free source for Indian markets. Requires Zerodha account.
★
NSE / BSE API (FREE)
Options chain, FII/DII data, max pain calculation. Scrape nseindia.com or use unofficial JSON endpoints.
$
Sensibull / Opstra API (PAID)
Clean OI data, PCR, gamma levels, IV surface. ~₹2000/mo. Worth it for options intelligence layer.
$
Polygon.io / Alpha Vantage (PAID/FREE)
US markets (S&P, Nasdaq), DXY, Crude, Bond yields. Free tier covers daily; paid for real-time.
★
NewsAPI / GNews (FREE)
Financial news feed. Feed through FinBERT NLP model for sentiment scoring (-1 to +1).
🧠 Signal Engine Logic
1
Technical Score (40% weight)
RSI + MACD + EMA alignment + VWAP position + Support/Resistance proximity. Outputs -100 to +100.
2
Options Score (35% weight)
PCR deviation from neutral + OI buildup direction + Max pain delta + VIX regime. Smart money proxy.
3
Global + Sentiment Score (25%)
SGX Nifty gap + DXY inverse correlation + Crude impact + FinBERT news score aggregated.
⚠
Event Filter (Hard Gate)
RBI/Fed dates → Force NO TRADE. VIX > 22 → Reduce size 50%. Expiry day < 1H left → No new entries.
